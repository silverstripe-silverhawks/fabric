# Fabricator

## Requirements
- PHP 7.4
- SilverStripe Framework ^4

## Uses
 - Vue 3 + Vite
 - Pinota

## Setup

`composer require silverstripe/fabricator`

Then add the extension to your project:
```
PageController:
  extensions:
    - SilverStripe\Fabricator\Extension\FabricatorExtension
```
